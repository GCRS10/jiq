# Linked Lists

## Reverse a Linked List
Reverse a singly linked list, iteratively, in O(1) extra space.

**File:** `src/main/java/com/interview/linkedlist/ReverseLinkedList.java`

## Detect a Cycle
Determine whether a linked list contains a cycle, using O(1) extra space
(Floyd's Tortoise and Hare algorithm).

**File:** `src/main/java/com/interview/linkedlist/DetectCycle.java`

## Merge Two Sorted Lists
Merge two sorted linked lists into a single sorted list by splicing together
their existing nodes.

**File:** `src/main/java/com/interview/linkedlist/MergeSortedLists.java`

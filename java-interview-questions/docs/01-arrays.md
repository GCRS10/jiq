# Arrays

## Two Sum
Given an array of integers `nums` and an integer `target`, return the indices
of the two numbers that add up to `target`. Assume exactly one valid answer
exists, and you may not use the same element twice.

**File:** `src/main/java/com/interview/arrays/TwoSum.java`

## Maximum Subarray (Kadane's Algorithm)
Given an integer array, find the contiguous subarray with the largest sum and
return that sum.

**File:** `src/main/java/com/interview/arrays/MaxSubArray.java`

## Rotate Array
Rotate an array to the right by `k` steps, in place, without allocating a
second array.

**File:** `src/main/java/com/interview/arrays/RotateArray.java`
